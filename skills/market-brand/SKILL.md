# ブランドボイス分析とガイドライン生成

## スキルの目的
利用可能なすべてのチャネルを横断してブランドのボイス・トーン・メッセージングを分析し、包括的なブランドボイスガイドラインドキュメントを生成します。このスキルはブランドのコミュニケーション方法を調査し、パターンと不一致を特定し、ライターやマーケターがブランドの一貫性を維持するために従えるアクション可能なガイドラインを作成します。

## 使用するタイミング
- ユーザーがブランドのボイスを理解またはドキュメント化したい場合
- ユーザーがチーム、フリーランサー、または代理店のためのブランドボイスガイドラインが必要な場合
- ユーザーがマーケティングチャネル全体の一貫性を確保したい場合
- ユーザーがリブランドまたはブランドアイデンティティを洗練させている場合
- ユーザーが自分のブランドボイスを競合他社と比較したい場合
- `/market brand <url>` または `/market brand` でトリガーされた場合

## 実行方法

### ステップ1: 素材の収集
ブランドのボイスを分析するために、複数のソースからコンテンツを調査します。以下の順に優先します：

**主要ソース（必ず分析する）：**
1. **ホームページ** -- ブランドの最も厳選された表現
2. **アバウトページ** -- ブランドが自己をどのように説明しているか
3. **製品/サービスページ** -- オファリングをどのように提示しているか

**二次ソース（利用可能な場合は分析する）：**
4. **ブログ記事**（最近の記事3〜5本）
5. **SNSプロフィール**（プロフィール文、最近の投稿、エンゲージメントスタイル）
6. **メールニュースレター**（ウェルカムメール、最近の配信）
7. **顧客向けコピー**（エラーメッセージ、オンボーディングフロー、ヘルプドキュメント）

**三次ソース：**
8. **求人票** -- 内部の文化と価値観を明らかにする
9. **プレスリリース** -- 公式なコミュニケーションスタイル
10. **広告コピー** -- 有料メッセージングのアプローチ
11. **動画スクリプトまたはポッドキャストの文字起こし** -- 話し言葉のブランドボイス

ウェブコンテンツへのアクセスにはブラウザツールまたはanalyze_page.pyスクリプトを使用します。SNSについては、ウェブサイトのソーシャルリンクを確認してリンクされたプロフィールを分析します。

### ステップ2: ボイスディメンション分析
4つの主要ディメンションに沿ってブランドのボイスをマッピングします。各ディメンションは二択ではなく、スペクトルです。

#### ディメンション1：フォーマル ←→ カジュアル
ブランドはフォーマリティのスペクトルのどこにいるか？

| シグナル | フォーマル | カジュアル |
|---|---|---|
| 短縮形 | 避ける（「do not」「cannot」） | 自由に使う（「don't」「can't」） |
| 文構造 | 複雑で長い文 | 短く、パンチのある文 |
| 語彙 | プロフェッショナル、業界標準 | 会話的、日常的な言葉 |
| 挨拶 | 「お客様各位」 | 「こんにちは！」 |
| 代名詞 | 三人称（「弊社」「一般に」） | 一人称/二人称（「私たち」「あなた」） |
| ユーモア | まれまたはなし | 頻繁、自然 |
| スラング/口語 | 決して使わない | 時々または頻繁に使う |

**スコア：1（非常にフォーマル）〜10（非常にカジュアル）**

**証拠が必要：** 評価を裏付ける素材からの3〜5の具体的な例を引用します。

#### ディメンション2：真面目 ←→ 軽快
ブランドはコミュニケーションにどれだけ軽さを加えているか？

| シグナル | 真面目 | 軽快 |
|---|---|---|
| トーン | 権威的、測定された | 明るい、楽しい |
| メタファー | まれ、保守的 | クリエイティブ、意外性がある |
| 感嘆符 | まれ | 頻繁 |
| 絵文字の使用 | 決して使わない | 時々または頻繁に使う |
| ダジャレ/言葉遊び | 決してしない | 楽しんでいる |
| エラーメッセージ | 「エラーが発生しました」 | 「おっと！何かがおかしくなりました」 |
| 自己批判 | 決してしない | 時々する |

**スコア：1（非常に真面目）〜10（非常に軽快）**

#### ディメンション3：テクニカル ←→ シンプル
ブランドはオーディエンスにどれだけのドメイン知識を前提としているか？

| シグナル | テクニカル | シンプル |
|---|---|---|
| 専門用語 | 業界用語を自由に使用 | 専門用語を避けるか説明する |
| 頭字語 | 定義なしで使用 | 初出時に展開する |
| 詳細レベル | 詳細な説明 | ハイレベルな概要 |
| オーディエンスの前提 | 専門家オーディエンス | 一般オーディエンス |
| データ/統計 | 頻繁で詳細 | 時々、簡略化 |
| 例 | 複雑、ドメイン固有 | シンプルで親しみやすい例え |

**スコア：1（非常にテクニカル）〜10（非常にシンプル）**

#### ディメンション4：控えめ ←→ 大胆
ブランドはどれだけのパーソナリティと自信を示しているか？

| シグナル | 控えめ | 大胆 |
|---|---|---|
| 主張 | ヘッジされている（「私たちは信じる」「役立つかもしれない」） | 直接的（「私たちは保証する」「最高の」） |
| 意見 | 中立、バランスのとれた | 強い、意見がある |
| 競合他社への言及 | 言及を避ける | 直接比較する |
| パーソナリティ | プロフェッショナル、おとなしい | 特徴的で記憶に残る |
| 約束 | 保守的 | 野心的 |
| 論争 | 避ける | 価値観と一致する場合は受け入れる |

**スコア：1（非常に控えめ）〜10（非常に大胆）**

### ステップ3: トーンスペクトラムのマッピング

4つのディメンションを超えて、ブランドのトーンがさまざまなコンテキストでどのように変化するかをマッピングします：

| コンテキスト | 典型的なトーン | 例 |
|---|---|---|
| ホームページ | [自信あり/歓迎/緊急/など] | 「[ホームページからの引用]」 |
| 製品説明 | [情報的/説得的/テクニカル/など] | 「[引用]」 |
| ブログ記事 | [教育的/会話的/権威的/など] | 「[引用]」 |
| SNS | [カジュアル/エンゲージング/プロモーショナル/など] | 「[引用]」 |
| エラー/404ページ | [謝罪的/ユーモラス/親切/など] | 「[引用]」 |
| メール件名 | [直接的/好奇心/緊急/など] | 「[引用]」 |
| CTAボタン | [行動指向/利益主導/緊急/など] | 「[引用]」 |
| カスタマーサポート | [共感的/プロフェッショナル/フレンドリー/など] | 「[引用]」 |

### ステップ4: ブランドパーソナリティフレームワーク

5つのコアパーソナリティアーキタイプのうち1つにブランドをマッピングします（ブランドは1〜2つをブレンドする場合があります）：

#### 5つのアーキタイプ

**1. オーソリティ（権威）**
- 特徴：専門家的、信頼性が高い、データドリブン、確立されている
- ボイス：自信があるが傲慢でない、教育的、正確
- 業界：金融、医療、B2Bエンタープライズ、法律、コンサルティング
- ブランド例：マッキンゼー、IBM、メイヨークリニック
- キーフレーズ：「調査によると...」「専門家が...」「業界をリードする...」

**2. イノベーター（革新者）**
- 特徴：先進的、破壊的、ビジョナリー、テクノロジーに精通
- ボイス：エキサイティング、未来志向、時に挑発的
- 業界：テクノロジー、SaaS、スタートアップ、再生可能エネルギー
- ブランド例：テスラ、Stripe、Notion
- キーフレーズ：「再想像する...」「未来の...」「私たちは構築している...」

**3. フレンド（友人）**
- 特徴：温かい、親しみやすい、親切、共感できる
- ボイス：会話的、共感的、インクルーシブ、励ます
- 業界：コンシューマー製品、教育、コミュニティプラットフォーム
- ブランド例：Mailchimp、Slack、Duolingo
- キーフレーズ：「分かります...」「あなたならできます...」「お手伝いします...」

**4. リベル（反逆者）**
- 特徴：大胆、慣習への挑戦、反骨精神、情熱的
- ボイス：直接的、意見が強い、時に対立的、記憶に残る
- 業界：ライフスタイル、フィットネス、クリエイティブ産業、D2C
- ブランド例：Nike、Oatly、Cards Against Humanity
- キーフレーズ：「もう妥協するのをやめよう...」「真実は...」「私たちは...に終止符を打つ」

**5. ガイド（案内人）**
- 特徴：賢い、辛抱強い、系統的、信頼できる
- ボイス：明確、指導的、支持的、知識豊富
- 業界：教育、専門能力開発、ツール、プラットフォーム
- ブランド例：HubSpot、Khan Academy、Ahrefs
- キーフレーズ：「こうやって...」「ステップバイステップで...」「完全ガイド...」

**評価：**
- プライマリアーキタイプ：[どれか、なぜか]
- セカンダリアーキタイプ：[該当する場合]
- アーキタイプの適合性：[強/中/弱 -- ブランドはこのアーキタイプをどれだけ体現しているか？]

### ステップ5: 語彙分析

ブランドの言葉遣いのパターンを特定します：

#### 頻繁に使う言葉
すべての素材を分析し、最も特徴的な15〜20の言葉またはフレーズを特定します。カテゴリー別に整理します：

**アクションワード：**（好んで使う動詞）
- 例：「構築する」「スケールする」「変革する」「効率化する」

**説明語：**（使用する形容詞）
- 例：「パワフルな」「シンプルな」「エンタープライズグレードの」「手間いらずの」

**価値語：**（価値観を反映する言葉）
- 例：「透明な」「持続可能な」「インクルーシブな」「革新的な」

**業界特有の用語：**
- 例：「ワークフロー」「パイプライン」「コンバージョン」「エンゲージメント」

#### 避ける言葉
特に欠如している、またはブランドに合わない言葉を特定します：

- ブランドにとってカジュアルすぎる言葉（フォーマルな場合）
- ブランドにとってテクニカルすぎる言葉（シンプルな場合）
- 意図的に避けている競合他社の用語
- 回避しているような業界の決まり文句

#### シグネチャーフレーズ
ブランドに繰り返し出現するフレーズ、タグライン、または言語パターンはあるか？

- タグライン：[持っている場合]
- 繰り返し出現するフレーズ：[気づいたパターン]
- 言語パターン：[例：常に動詞で文を始める、ダッシュを頻繁に使用、短い段落を好む]

### ステップ6: 競合他社のボイス比較

2〜3社の主要競合他社とブランドのボイスを比較します：

**ボイス比較マトリックス：**
| ディメンション | [ブランド] | 競合他社1 | 競合他社2 | 競合他社3 |
|---|---|---|---|---|
| フォーマル ↔ カジュアル | X/10 | X/10 | X/10 | X/10 |
| 真面目 ↔ 軽快 | X/10 | X/10 | X/10 | X/10 |
| テクニカル ↔ シンプル | X/10 | X/10 | X/10 | X/10 |
| 控えめ ↔ 大胆 | X/10 | X/10 | X/10 | X/10 |
| プライマリアーキタイプ | [タイプ] | [タイプ] | [タイプ] | [タイプ] |

**差別化の評価：**
- ブランドのボイスはどれだけ競合他社と異なるか？
- ボイスが重なっている部分は？（差別化の機会）
- 競合環境で占有されていないボイスの領域は何か？
- ボイスの差別化のための具体的な推奨事項

### ステップ7: 一貫性の監査

分析したすべてのチャネルでボイスの一貫性を評価します：

| チャネル | ボイスの一貫性 | メモ |
|---|---|---|
| ホームページ | 一貫/概ね一貫/不一致 | [具体的な観察] |
| アバウトページ | 一貫/概ね一貫/不一致 | [メモ] |
| ブログ | 一貫/概ね一貫/不一致 | [メモ] |
| SNS | 一貫/概ね一貫/不一致 | [メモ] |
| メール | 一貫/概ね一貫/不一致 | [メモ] |
| 製品ページ | 一貫/概ね一貫/不一致 | [メモ] |

**よくある一貫性の問題：**
- 異なるライターがはっきりと異なるトーンを生み出している
- SNSのボイスがウェブサイトと大きく異なる
- フォーマルなウェブサイトコピーだがカジュアルなメールニュースレター
- 製品ページとは全く異なるボイスで書かれたブログコンテンツ
- ブランドに合わないと感じるエラーメッセージやマイクロコピー
- 現在のブランドボイスに合わせて更新されていない古いページ

**全体的な一貫性スコア：** X/10

### ステップ8: ブランドメッセージング階層

最も凝縮されたものから最も展開されたものへ、ブランドのメッセージングを記録します：

#### レベル1：タグライン（10語未満）
ブランドメッセージの最も圧縮された形式。
- 現在：「[既存のタグライン、またはそれが示すもの]」
- 評価：コアな価値提案を捉えているか？

#### レベル2：バリュープロポジション（各1文）
ブランドの約束を支持する3〜5のコアな価値提案。
1. 「[価値提案1]」
2. 「[価値提案2]」
3. 「[価値提案3]」

#### レベル3：エレベーターピッチ（30秒 / 75語）
ブランドが何をするか、なぜそれが重要かの会話的な説明。
「[分析したコンテンツに基づくエレベーターピッチの下書き]」

#### レベル4：定型文（100〜150語）
プレスリリース、メール署名、スピーカープロフィールで使われる標準的な「会社概要」段落。
「[分析したコンテンツに基づく定型文の下書き]」

#### レベル5：完全なブランドストーリー（300〜500語）
ブランドが誰で、何を支持し、なぜ存在するかについての完全な物語。
- 現状：[存在する/一部/なし]
- 改善のための推奨事項

### ステップ9: ブランドボイスドキュメンテーションの生成

```
[特徴2]                          [反特徴2]

[特徴3]                          [反特徴3]
例：「明確」                       例：「過度に単純化」

[特徴4]                          [反特徴4]
例：「大胆」                       例：「攻撃的」
```

#### ライティングのDo's and Don'ts

**すること：**
- [分析に基づく具体的なライティング指示]
- [例：「自然に聞こえるように短縮形を使う（私たちは、あなたは、それは）」]
- [例：「機能ではなく利益からリードする」]
- [例：「すべての見出しとCTAに能動態を使用する」]
- [例：「『あなた』と『あなたの』で読者に直接話しかける」]

**しないこと：**
- [分析に基づく具体的なアンチパターン]
- [例：「専門用語を説明せずに使わない」]
- [例：「CTAに受動態を使わない」]
- [例：「1段落に1回以上感嘆符を使わない」]
- [例：「『私たちは』で文を始めない -- 顧客に焦点を当てる」]

## 主要原則
- ブランドボイス分析は探偵のように読むことが必要です。すべての言葉の選択、句読点の決断、文構造がブランドがどのように見られたいかについて何かを明らかにしています。
- すべての評価に**証拠**を提供します。「ブランドはカジュアルだ」と言うだけでなく、それを証明する具体的な例を引用します。
- ブランドボイスガイドは、そのブランドと一度も仕事をしたことのない人でも使えるものであるべきです。新しいコピーライターはこのドキュメントを読んでブランドに沿ったコンテンツを書けるはずです。
- コピーサンプルは成果物の最も価値ある部分です。人々は説明ではなく例でボイスを学びます。ライターがすべてのコンテキスト（見出し、本文コピー、SNS、メール、エラーメッセージ）で参照できるよう、サンプルを多様にします。
- ボイスとトーンは異なります。ボイスは一貫したパーソナリティです。トーンはコンテキストによって変化します（クレーム対応は製品ローンチアナウンスとは異なりますが、両方とも同じボイスであるべきです）。
- ブランドのボイスがチャネル全体で一貫していない場合は、失敗としてではなく、ブランドを強化する機会としてフレーミングします。一貫性の問題は一般的で修正可能です。
- ユーザーが以前に `/market competitors` を実行していた場合は、そのデータを競合他社のボイス比較セクションに使用します。
- ボイスのディメンションはビジュアル的に（テキストベースのスペクトルとして）プロットし、ステークホルダーが一目でポジショニングを理解できるようにします。
